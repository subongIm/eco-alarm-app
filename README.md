# Eco Alarm

Flutter 기반 알람 앱입니다.

## 주요 기능

- 알람 생성, 수정, 삭제
- 반복 알람 설정 (요일별)
- 스누즈 기능
- 진동 및 커스텀 사운드 지원
- 플랫폼별 최적화 (Android/iOS)

## 기술 스택

- Flutter
- Riverpod (상태 관리)
- Hive (로컬 데이터베이스)
- flutter_local_notifications (로컬 알림)
- android_alarm_manager_plus (Android 알람)

## 사운드 파일 저작권

이 프로젝트에 포함된 알람 사운드 파일들은 [Mixkit](https://mixkit.co/)에서 제공하는 무료 사운드 효과입니다.

### Mixkit 라이선스
- Mixkit에서 제공하는 사운드 파일은 무료로 사용 가능하며 상업적 사용도 허용됩니다.
- 출처 표기는 필수가 아닙니다.
- 자세한 라이선스 정보는 [Mixkit 라이선스 페이지](https://mixkit.co/license/)를 참조하세요.

### 사용된 사운드 파일
- `mixkit-classic-alarm-995.wav`
- `mixkit-morning-clock-alarm-1003.wav`
- `mixkit-short-rooster-crowing-2470.wav`
- `mixkit-horde-of-barking-dogs-60.wav`
- `mixkit-magic-festive-melody-2986.wav`
- `mixkit-wrong-long-buzzer-954.wav`

**면책 조항**: 이 프로젝트의 개발자는 Mixkit에서 제공하는 사운드 파일의 저작권을 주장하지 않으며, 모든 저작권은 Mixkit 및 해당 사운드 파일의 원 저작자에게 있습니다. 사운드 파일 사용에 대한 모든 책임은 사용자에게 있습니다.
